# 🤖 AI Mock Interviewer

An **AI-powered mock interview platform** that simulates real interview environments with
role-based questions, coding challenges, and intelligent evaluation — built using the **MERN stack**.

This project is designed for **students and developers** to practice interviews in a secure,
proctored, and realistic setup.

---

## 🚀 Features

- 🎯 Role & experience based mock interviews  
- 🧠 AI-generated questions and evaluations (Gemini API)  
- 📝 Multiple answer types:
  - MCQs
  - Text answers
  - Audio answers
  - Coding challenges
- 💻 Built-in coding environment
- 🎥 Camera & microphone based proctoring
- 🚫 Tab switching & fullscreen enforcement
- 📊 Interview history & performance reports
- 🔐 Secure authentication system

---

## 🛠 Tech Stack

### Frontend
- React (Vite)
- Tailwind CSS
- shadcn/ui
- Zustand
- Axios

### Backend
- Node.js
- Express.js
- MongoDB (Mongoose)
- Socket.IO
- JWT Authentication

### AI Layer
- Google Gemini API (AI Studio)
- Provider-agnostic AI adapter (easy to swap later)
